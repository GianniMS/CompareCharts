# CompareCharts
This example shows how to compare two sets of ratings (like user vs. author) using overlaid radar charts in React Native.

Features:

Radar chart with labeled metrics

Overlay to compare two ratings

Toggle comparison visibility

Tech:

React Native

react-native-gifted-charts

Usage:

Install: npm install react-native-gifted-charts

Copy the RatingComparison component

Customize the data and use in your app

Great for review apps, feedback tools, or personal tracking.
